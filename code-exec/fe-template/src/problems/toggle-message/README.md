# Toggle Message Component

## Problem Description
Create a React component that contains a button and a message. The message "Hello World" should initially be hidden. Clicking the button should toggle the visibility of the message.

## Requirements
1.  **Button Label:** The button should say "Show Message" when the message is hidden, and "Hide Message" when the message is visible.
2.  **Message:** The message text is "Hello World".
3.  **Interaction:** Clicking the button toggles the message's presence in the DOM (conditional rendering).
